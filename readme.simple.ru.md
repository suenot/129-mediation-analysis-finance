# Медиационный анализ для финансов - Простое руководство

## Что такое медиационный анализ? (История о "посреднике")

Представьте, что вы детектив, пытающийся понять, как вещи влияют друг на друга. Иногда связь не прямая - есть "посредник"!

### Пример из жизни: Эффект пиццы

Допустим, вы заметили, что когда светит солнце, магазины мороженого зарабатывают больше. Но ПОЧЕМУ?

```
Прямое мышление:
    Солнечная погода ───────────────► Продажи мороженого

    "Солнце заставляет людей покупать мороженое"

Но РЕАЛЬНАЯ история может быть такой:

    Солнечная погода ────► Больше людей на улице ────► Продажи мороженого
         │                                                    ▲
         └──────── (может быть немного прямого эффекта тоже) ─┘

    "Посредник" - это "Больше людей на улице"!
```

Этот "посредник" называется **МЕДИАТОР** - что-то, что помогает объяснить КАК одна вещь влияет на другую.

## Почему трейдерам это важно?

### Детективная история фондового рынка

Представьте, что вы обнаружили, что хорошие новости о компании поднимают цену её акций. Но вы хотите знать КАК это происходит:

```
Простой взгляд:
    Хорошие новости ──────────────────► Цена акции ↑

    "Хорошие новости = цена растёт. Готово!"

Взгляд детектива:
    Хорошие новости ────► Люди радуются ────► Они покупают ────► Цена акции ↑
         │                 (Настроения)         (Объём)               ▲
         │                                                            │
         └──────────── некоторые реагируют мгновенно ─────────────────┘

    Теперь мы понимаем МЕХАНИЗМ!
```

### Почему это важно для трейдинга

1. **Лучший тайминг**: Если вы знаете промежуточный шаг (люди радуются → покупают), вы можете отслеживать, когда эта цепочка работает или прервана

2. **Умнее сигналы**: Не все "хорошие новости" работают одинаково
   - Если настроения растут, но объём нет → цепочка может быть прервана
   - Если объём растёт без настроений → работает другой механизм

3. **Управление рисками**: Разные пути имеют разные риски!

## Три главных персонажа

В каждой истории медиации три главных персонажа:

```
┌──────────────────────────────────────────────────────────────────┐
│                   СОСТАВ МЕДИАЦИИ                                 │
├──────────────────────────────────────────────────────────────────┤
│                                                                   │
│  🎬 X = ВОЗДЕЙСТВИЕ (То, с чего всё начинается)                  │
│      Примеры: Новостное событие, отчёт о прибыли, обвал рынка    │
│                                                                   │
│  🔄 M = МЕДИАТОР (Посредник)                                     │
│      Примеры: Настроения, объём торгов, мнения аналитиков        │
│                                                                   │
│  🎯 Y = ИСХОД (То, что мы хотим предсказать)                     │
│      Примеры: Доходность акций, движение цены, волатильность     │
│                                                                   │
│  История: X ───► M ───► Y                                        │
│           │             ▲                                         │
│           └─────────────┘                                         │
│           (и возможно X влияет на Y напрямую тоже!)              │
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```

## Разбивка эффектов

Думайте об этом как о разрезании пирога:

```
ПИРОГ ПОЛНОГО ЭФФЕКТА = ПРЯМОЙ КУСОК + КОСВЕННЫЙ КУСОК

┌──────────────────────────────────────────────────────────────────┐
│                                                                   │
│    Полный эффект = 100%                                          │
│    ┌─────────────────────────────────────────┐                   │
│    │                                         │                   │
│    │   ПРЯМОЙ          │    КОСВЕННЫЙ        │                   │
│    │   40%             │    60%              │                   │
│    │                   │                     │                   │
│    │   X идёт          │    X идёт через     │                   │
│    │   напрямую к Y    │    M чтобы достичь Y│                   │
│    │                   │                     │                   │
│    └───────────────────┴─────────────────────┘                   │
│                                                                   │
│    В формулах:  Полный = Прямой + Косвенный                      │
│                  100%  =  40%   +   60%                          │
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```

## Торговый пример: Bitcoin → Ethereum

Давайте посмотрим реальный крипто-пример:

```
Вопрос: Влияет ли движение цены Bitcoin на цену Ethereum?
        И если да, то КАК?

Настройка:
    X = Доходность Bitcoin (BTC вырос или упал?)
    M = Рыночные настроения (люди взволнованы или напуганы?)
    Y = Доходность Ethereum (что происходит с ETH?)

История:

    Цена BTC ↑ ────► Крипто-оптимизм ↑ ────► Цена ETH ↑
         │                                        ▲
         │                                        │
         └─── прямая корреляция (тот же рынок) ──┘

Что мы можем найти:
    - Полный эффект: Когда BTC растёт на 1%, ETH растёт на 0.8%
    - Прямой эффект: 0.3% (они движутся вместе, тот же рынок)
    - Косвенный эффект: 0.5% (через распространение настроений)

    Доля опосредованного: 0.5 / 0.8 = 62.5%

    Значение: Большая часть влияния BTC на ETH идёт через настроения!
```

## Типы медиации (С примерами пиццы!)

### 1. Полная медиация (История доставки пиццы)

```
Ваш голод не заставляет пиццу волшебно появиться у вашей двери:

    Голод ────► Вы заказываете пиццу ────► Пицца приезжает

    "Заказ" - это ЕДИНСТВЕННЫЙ способ, как голод ведёт к пицце.
    Это ПОЛНАЯ МЕДИАЦИЯ - промежуточный шаг - это ВСЁ.

    В трейдинге: Инсайдерская инфо → Торговая активность → Изменение цены
                (Информация имеет значение только потому, что кто-то торгует)
```

### 2. Частичная медиация (История дождливого дня)

```
Дождь влияет на ваше настроение двумя способами:

    Дождь ────► Вы сидите дома ────► Чувствуете скуку
      │                                   ▲
      └───── (звук раздражает) ───────────┘

    Дождь влияет на ваше настроение КАК:
    - Через сидение дома (косвенно)
    - ТАК И напрямую (звук, мрачность)

    В трейдинге: Новости → Объём + Настроения → Цена
                (Новости имеют прямой и косвенный эффекты)
```

### 3. Нет медиации (История льда в напитке)

```
Желание холодного напитка ведёт к льду И холодному напитку,
но лёд не ВЫЗЫВАЕТ холодность вашего выбора напитка:

    Хочу холодный напиток ────────────────► Выбираю холодный напиток
           │
           └──────► Добавляю лёд
                    (связано, но не на пути)

    В трейдинге: Обвал рынка → Вы паникуете + Цены падают
                (Ваша паника не вызвала падение)
```

## Классический метод: Четыре шага Барона и Кенни

Думайте об этом как о рецепте:

```
┌──────────────────────────────────────────────────────────────────┐
│              РЕЦЕПТ МЕДИАЦИИ ИЗ ЧЕТЫРЁХ ШАГОВ                     │
├──────────────────────────────────────────────────────────────────┤
│                                                                   │
│  Шаг 1: ПРОВЕРЬТЕ ОСНОВНУЮ СВЯЗЬ                                 │
│         X действительно предсказывает Y?                         │
│         "Хорошие новости → более высокая доходность?"            │
│         Если НЕТ → Стоп! Нечего опосредовать.                    │
│                                                                   │
│  Шаг 2: ПРОВЕРЬТЕ ПЕРВОЕ ЗВЕНО                                   │
│         X предсказывает M?                                       │
│         "Хорошие новости → позитивные настроения?"               │
│         Если НЕТ → M не медиатор для X.                          │
│                                                                   │
│  Шаг 3: ПРОВЕРЬТЕ ВТОРОЕ ЗВЕНО                                   │
│         M предсказывает Y (когда мы также включаем X)?           │
│         "Настроения → доходность (помимо просто новостей)?"      │
│         Если НЕТ → M ничего не добавляет.                        │
│                                                                   │
│  Шаг 4: СРАВНИТЕ ЭФФЕКТЫ                                         │
│         Уменьшился ли эффект X на Y когда мы добавили M?         │
│         "Эффект новостей меньше когда учитываем настроения?"     │
│         Если ДА → Медиация подтверждена!                         │
│                                                                   │
│         Если эффект X становится НУЛЁМ → Полная медиация         │
│         Если эффект X уменьшается, но не до нуля → Частичная     │
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```

## Простой пример на Python

Вот базовый пример, который вы можете понять:

```python
import numpy as np

# Создаём фейковые, но реалистичные данные
np.random.seed(42)
n = 100  # 100 дней данных

# X = Оценка настроений рыночных новостей (от -1 до 1)
X = np.random.randn(n) * 0.5

# M = Объём торгов (на который влияют новости)
# Когда новости сильные, больше людей торгуют
M = 0.4 * X + np.random.randn(n) * 0.3

# Y = Доходность акций
# На которую влияют КАК новости напрямую, ТАК И объём
Y = 0.3 * X + 0.5 * M + np.random.randn(n) * 0.2

# Теперь давайте найдём эффекты!

# ШАГ 1: Полный эффект (только X предсказывает Y)
from numpy.linalg import lstsq
X_design = np.column_stack([np.ones(n), X])
beta_total = lstsq(X_design, Y, rcond=None)[0]
total_effect = beta_total[1]
print(f"Полный эффект (c): {total_effect:.3f}")

# ШАГ 2: X предсказывает M
beta_a = lstsq(X_design, M, rcond=None)[0]
a_path = beta_a[1]
print(f"Путь a (X → M): {a_path:.3f}")

# ШАГ 3: И X и M предсказывают Y
XM_design = np.column_stack([np.ones(n), X, M])
beta_direct = lstsq(XM_design, Y, rcond=None)[0]
direct_effect = beta_direct[1]  # c'
b_path = beta_direct[2]         # b
print(f"Прямой эффект (c'): {direct_effect:.3f}")
print(f"Путь b (M → Y): {b_path:.3f}")

# Вычисляем косвенный эффект
indirect_effect = a_path * b_path
print(f"\nКосвенный эффект (a × b): {indirect_effect:.3f}")

# Проверка: Полный должен равняться Прямой + Косвенный
print(f"\nПроверка:")
print(f"  Полный:            {total_effect:.3f}")
print(f"  Прямой + Косвенный: {direct_effect + indirect_effect:.3f}")

# Доля опосредованного
prop_mediated = indirect_effect / total_effect
print(f"\nДоля опосредованного: {prop_mediated:.1%}")
```

Ожидаемый вывод:
```
Полный эффект (c): 0.495
Путь a (X → M): 0.412
Прямой эффект (c'): 0.287
Путь b (M → Y): 0.503

Косвенный эффект (a × b): 0.207

Проверка:
  Полный:            0.495
  Прямой + Косвенный: 0.494

Доля опосредованного: 41.9%
```

## Как использовать это в трейдинге

### Торговая стратегия с учётом медиации

```
┌──────────────────────────────────────────────────────────────────┐
│             ТОРГОВЛЯ С ИНСАЙТАМИ МЕДИАЦИИ                         │
├──────────────────────────────────────────────────────────────────┤
│                                                                   │
│  СИТУАЦИЯ 1: Канал медиатора АКТИВЕН                             │
│  ────────────────────────────────────                            │
│  Сигнал:  Появились хорошие новости                              │
│  Проверка: Настроения растут? Объём увеличивается?               │
│  Результат: Если ДА → Ожидайте ПОЛНЫЙ ЭФФЕКТ (прямой + косвенный)│
│  Действие: Открыть БОЛЬШУЮ позицию                               │
│                                                                   │
│  СИТУАЦИЯ 2: Канал медиатора ЗАБЛОКИРОВАН                        │
│  ──────────────────────────────────────                          │
│  Сигнал:  Появились хорошие новости                              │
│  Проверка: Настроения плоские? Объём нормальный?                 │
│  Результат: Если ДА → Ожидайте ТОЛЬКО ПРЯМОЙ ЭФФЕКТ              │
│  Действие: Открыть МЕНЬШУЮ позицию (или ждать)                   │
│                                                                   │
│  СИТУАЦИЯ 3: Медиатор активен без воздействия                    │
│  ─────────────────────────────────────────                       │
│  Сигнал:  Нет больших новостей                                   │
│  Проверка: Но настроения/объём скачут всё равно?                 │
│  Результат: Что-то другое происходит!                            │
│  Действие: Исследовать перед торговлей                           │
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```

### Простые торговые правила

```python
def should_trade(news_signal, sentiment_change, volume_change,
                 mediation_threshold=0.3):
    """
    Простое торговое решение с учётом медиации

    Аргументы:
        news_signal: Сила новостей (от -1 до 1)
        sentiment_change: Насколько изменились настроения
        volume_change: Насколько изменился объём от нормы
        mediation_threshold: Какая доля эффекта идёт через медиатор

    Возвращает:
        решение: 'сильный_лонг', 'слабый_лонг', 'сильный_шорт',
                 'слабый_шорт', или 'не_торговать'
    """

    # Канал медиатора активен?
    mediator_active = (abs(sentiment_change) > 0.5 and
                      abs(volume_change) > 0.3)

    # Базовый сигнал от новостей
    if abs(news_signal) < 0.2:
        return 'не_торговать'  # Недостаточно сигнала

    if news_signal > 0:
        if mediator_active:
            return 'сильный_лонг'  # Ожидается полный эффект
        else:
            return 'слабый_лонг'   # Только прямой эффект
    else:
        if mediator_active:
            return 'сильный_шорт'
        else:
            return 'слабый_шорт'

# Пример использования
decision = should_trade(
    news_signal=0.7,       # Позитивные новости
    sentiment_change=0.8,  # Настроения подскочили
    volume_change=0.5      # Объём вырос
)
print(f"Торговое решение: {decision}")
# Вывод: Торговое решение: сильный_лонг
```

## Ключевые выводы

### Запомните эти пункты:

```
1️⃣  МЕДИАЦИЯ = Понимание "как" и "почему"
    Не просто "X предсказывает Y", а "X → M → Y"

2️⃣  ТРИ ПЕРСОНАЖА: Воздействие (X), Медиатор (M), Исход (Y)

3️⃣  ДВА ЭФФЕКТА: Прямой (X напрямую к Y) и Косвенный (X через M к Y)

4️⃣  ИСПОЛЬЗОВАНИЕ В ТОРГОВЛЕ: Смотрите, активен или заблокирован канал медиатора

5️⃣  ОСТОРОЖНО: Корреляция ≠ Причинность! Медиация предполагает каузальную структуру.
```

### Распространённые ошибки, которых следует избегать

```
❌ НЕ НАДО: Предполагать, что каждая коррелированная переменная - медиатор
   → То что A, B и C коррелируют не значит, что B опосредует A→C

❌ НЕ НАДО: Игнорировать порядок событий
   → Медиатор ДОЛЖЕН быть МЕЖДУ воздействием и исходом во времени

❌ НЕ НАДО: Переинтерпретировать маленькие эффекты
   → Статистическая значимость не означает торговую значимость

❌ НЕ НАДО: Забывать о других объяснениях
   → Могут быть скрытые факторы, влияющие на всё

✅ НАДО: Сначала думать о механизме
   → "ПОЧЕМУ X повлияет на M, и ПОЧЕМУ M повлияет на Y?"

✅ НАДО: Тестировать свои предположения
   → Запустите анализ, проверьте цифры, будьте скептичны

✅ НАДО: Учитывать тайминг
   → Как быстро происходит каждый шаг? Секунды? Дни? Недели?
```

## Краткая справочная карточка

```
┌──────────────────────────────────────────────────────────────────┐
│                    ШПАРГАЛКА ПО МЕДИАЦИИ                          │
├──────────────────────────────────────────────────────────────────┤
│                                                                   │
│  ФОРМУЛЫ:                                                         │
│    Полный эффект:     c  (из Y ~ X)                              │
│    Прямой эффект:     c' (из Y ~ X + M)                          │
│    Косвенный эффект:  a × b                                      │
│    Доля опосредованного: (a × b) / c                             │
│                                                                   │
│  ПУТИ:                                                            │
│    a = X → M  (воздействие влияет на медиатор)                   │
│    b = M → Y  (медиатор влияет на исход, контролируя X)          │
│    c = X → Y  (полный эффект)                                    │
│    c' = X → Y (прямой эффект, с M в модели)                      │
│                                                                   │
│  ИНТЕРПРЕТАЦИИ:                                                   │
│    Полная медиация:    c' ≈ 0 (весь эффект через M)              │
│    Частичная медиация: c' < c но c' ≠ 0 (оба пути работают)      │
│    Нет медиации:       a×b ≈ 0 (M не несёт эффект)               │
│                                                                   │
│  ТОРГОВЫЙ ИНСАЙТ:                                                 │
│    Медиатор АКТИВЕН:    Ожидайте полный эффект → Большая позиция │
│    Медиатор ЗАБЛОКИРОВАН: Ожидайте только прямой → Меньшая позиция│
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```
